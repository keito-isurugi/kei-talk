### display: grid;を使用


```css
.container {
  display: grid;
  place-items: center; 
  /* 
	  上下のみ中央揃えにしたい場合
	  place-items: center　start; 
  　*/
}
```


### display: flex;を使用


```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
```
